create c clock udp ~hostname ip ~port ~truncate () is reporter, which sends log messages to ip, port via UDP. Upon failure, a message is emitted to the console c. Each message can be truncated: truncate defaults to 65535 bytes. The hostname is part of each syslog message. The port defaults to 514. facility is the default syslog facility (see Logs_syslog.message).
